# Visual Direction - Portfolio Raúl Ortega

## Dirección general

La estética del sitio será:

Profesional editorial + soft-tech sobria.

La web debe comunicar confianza, orden, amabilidad, criterio técnico y cuidado visual, sin sentirse demasiado corporativa, demasiado artística, demasiado startup o demasiado genérica.

## Sensación buscada

La persona que vea la web debería pensar:

“Es ordenado, confiable, tiene potencial, cuida los detalles y puede integrarse bien a un equipo.”

## Prioridad visual

1. Presentación profesional clara.
2. Experiencia y proyectos como CV extendido.
3. Detalles visuales modernos, pero controlados.

## Evitar

- Landing SaaS genérica.
- Página hecha completamente por IA.
- Exceso de cápsulas.
- Exceso de glassmorphism.
- Web demasiado oscura.
- Web demasiado artística o anime.
- Web sobrecargada.

## Color

El azul será un acento controlado, no un color dominante en todos los elementos.

Regla aproximada:

- 70% neutros claros.
- 20% texto y estructura.
- 10% acento azul.

## Paleta base

- Fondo principal: blanco roto cálido.
- Superficies: blanco o casi blanco.
- Texto principal: navy/casi negro.
- Texto secundario: gris azulado.
- Azul principal: acento para botones, links y detalles.
- Verde grisáceo: apoyo secundario muy sutil.

## Formas

Reducir el exceso de formas tipo cápsula.

Usar bordes redondeados moderados, más editoriales y adultos.

## Glassmorphism

Usar con moderación.

Preferir superficies sólidas, bordes finos y sombras suaves.
